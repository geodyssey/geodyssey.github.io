Hipparchus Free Demo Downloads

Following is a collection of freely downloadable Hipparchus Windows demo suites.

Their format is generally that of a self-extracting zipped collection of Windows directories, programs and files. None of these downloads need to be formally "installed" within Windows. They are completely self-contained and may be removed at any time with a simple Windows delete operation.

Galileo for Windows

Four downloadable Galileo suites are:

  • Galileo Basics is the basic suite of Galileo programs, control files, scripts and associated data. This suite is approximately 2.7MB in length (zipped). After downloading, it is usually best to be unzipped into a root directory named Galileo. Download Galileo Basics now.

  • Galileo in Space is a Galileo add-in suite of scripts and data demonstrating the modeling of Earth (or other planetary) orbits and the reconciliation of their positions in space with what their sensors "see" on the planetary surface. This suite is approximately 4.6MB in length (zipped). After downloading, it should be unzipped into the same directory as that chosen for the Galileo Basics. Download Galileo in Space now.

  • Galileo in 3D is a Galileo add-in suite of scripts and data demonstrating the modeling of 3D ellipsoidal surfaces (elevations, depths, etc.). This suite is approximately 13.3MB in length (zipped). After downloading, it should be unzipped into the same directory as that previously chosen for the Galileo Basics. Download Galileo in 3D now.

  • Galileo USA is a Galileo add-in suite of scripts and data demonstrating detailed computations and presentations of objects in the USA. This suite is approximately 25.2MB in length (zipped). After downloading, it should be unzipped into the same directory as that chosen for the Galileo Basics. Download Galileo USA now.

Georama

One downloadable Georama "sampler suite" features a regional Atlas, namely New Zealand. This suite is approximately 4MB in length (zipped). After downloading, it is usually best to be unzipped into a root directory named Georama. Download Georama for New Zealand now.

The complete Georama CD World and USA atlas is also available for download as a 450 MB CD image (".iso") file. It's md5sum is 4fccfea77e6d4d7a20a1353332c194e7 georama.iso. Download Georama CD .iso file now.
