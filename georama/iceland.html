A Georama World Atlas extract covering Iceland is now available for download!

Georama is a Windows application CD-ROM that implements an extensible World Atlas. This free geographical extract offers an Atlas of Iceland. It is a fully functioned version of Georama for Windows , corresponding to Hipparchus Release 3.2.

Georama is written in 32-bit ANSI C and uses the Win32 GUI API directly. No other GUI components are required. It accesses its files using C/C++ stream I/O only. No DBMS is required. All components of the original Georama CD-ROM are compatible with this version.

Before downloading Georama (Iceland Extract), you may wish to view its extensive Georama Guide. (This is also a part of the download).

The download is a self-extracting zip file of approximately 1.5MB. Just download iceland.exe into a temporary directory, create a new directory (best called georama)and execute iceland.exe into this directory.
