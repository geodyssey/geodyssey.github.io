Introduction

Welcome to the Catalog of Prebuilt Hipparchus Voronoi Index Structures.

This catalog describes a number of Voronoi indexes that have been prebuilt for your use in the development of general Hipparchus-based applications or in the development of new Atlases for Georama.

The structures catalogued here are freely available for download from the web. Size information and a download option are presented with each structure.

Each structure is available in two forms:

  • a binary file accessible by Georama, Galileo and any Hipparchus-based application designed for Win32 compatibility (i.e., Windows 95/98/2000/NT). Such files carry the filename extension .vix.

  • an ASCII file of cell centerpoint latitude/longitude coordinates, suitable for pre-processing with the cross-platform Hipparchus Utility program CELLWVS. Such files carry the filename extension .pts.

If you have sufficient bandwidth, you might consider downloading the entire set of Voronoi indexing structures offered here. Otherwise, please browse this catalog, selecting only those structures that best fit your application.
